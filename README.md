# busca-binaria-java

Este projeto contém uma implementação do algoritmo de busca binária em Java, baseado no livro "Entendendo Algoritmos".

## 📌 O que aprendi
- Como funciona a busca binária
- Uso de índices (baixo, alto, meio)
- Complexidade O(log n)

## 🚀 Tecnologias
- Java

## 💻 Como funciona
O algoritmo divide a lista ordenada pela metade a cada passo até encontrar o elemento desejado.
